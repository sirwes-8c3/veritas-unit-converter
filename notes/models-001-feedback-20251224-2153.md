### [BLOCKER] Compilation Error
* **File:** `VeritasUnitConverter/Models/Unit.swift`
* **Issue:** The struct implements a custom `init(from:)` decoder but does not define `enum CodingKeys`. Swift does not synthesize `CodingKeys` when a custom decoder is present, which will cause the build to fail at `try decoder.container(keyedBy: CodingKeys.self)`.
* **Fix:** Add the `CodingKeys` enum to `Unit.swift`:

```swift
enum CodingKeys: String, CodingKey {
    case id, name, symbol, toBase, toBaseOffset
}
```
